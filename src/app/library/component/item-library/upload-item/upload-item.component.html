Item

What It Does

Items are things. Things are called items in Clearly Inventory. Why? Because a lot of common terms in inventory sound weird if you use the word thing instead of item. For example, "item description" is a familiar term, but "thing description" just sounds odd.

When to use its

The 'Item' upload is ideal for populating your Item Library for the first time, or if you wish to add new Item to the Item Library.


What's Required

The layout and organization of the file that you upload must exactly match our template file, and you can perform a maximum of 2,000 events (rows) per upload.

Location

What It Does

Location is a physical place where units in inventory can be found. Units cannot exist in two locations at the same time. Locations can be as general or specific as you like. Good location names should inform people where the units can be found at a given time.

When to use its

The 'Location' upload is ideal for populating your Location Library for the first time, or if you wish to add new Location to the Location Library.
